# LeetCode Hint Helper

A lightweight Chrome extension that adds step-by-step hints to LeetCode problems using a clean floating UI. It supports 20+ popular problems and displays hints progressively through “Show Hint” and “Next Hint”.

## Features
- Floating “Show Hint” button on LeetCode problem pages  
- Progressive hint system  
- Covers 20+ common problems (arrays, strings, stacks, DP, etc.)  
- Built with Manifest V3  
- Fully client-side and easy to extend  

## Project Structure
```
leetcode-hint-helper/
 ├── manifest.json
 ├── content.js
 └── styles.css
```

## Installation (Developer Mode)
1. Download or clone this repository  
2. Go to `chrome://extensions/`  
3. Enable **Developer mode**  
4. Click **Load unpacked**  
5. Select the `leetcode-hint-helper` folder  

## Tech Stack
JavaScript, HTML/CSS, Chrome Extensions (MV3)
